# Al Brooks 价格行为学 — 交易规则体系

> 来源：Al Brooks《Trading Price Action》三部曲、LouiePriceAction YouTube 频道 120+ 视频、NexusFi 社区、中文学习笔记。

## 一、核心理念

### 交易员公式
```
期望值 = 胜率 × 盈利 — 失败概率 × 亏损
```
- 三个变量同等重要：风险、收益、概率
- 除非收益至少等于风险，否则不值得做
- 盈亏比 1:1 → 至少 60% 胜率才能盈利
- 盈亏比 2:1 → 40% 胜率即可盈利

### 40-60 定律
- 市场 90% 时间处于通道或震荡，多空概率在 40%-60% 之间
- 只有 10% 时间是强趋势/强突破，概率偏移到 60%+
- **管理交易比入场更重要** — 90% 时间赚钱靠交易管理，不靠入场

### 核心决策框架
```
高时间级别 → 判断背景（趋势 / 通道 / 震荡区间）
低时间级别 → 寻找可执行入场形态
每次入场前必须有至少两个理由
```

## 二、市场周期（Market Cycle）

| 阶段 | 特点 | 交易策略 |
|------|------|---------|
| **突破（Breakout）** | K线几乎无重叠，无回调或仅1-2根 | 顺势市价入场，小仓位，止损在起涨K线低点 |
| **窄通道（Tight Channel）** | 回调1-3根K线，HTF看是突破 | 只做顺势，回调买入/反弹卖出 |
| **宽通道（Broad Channel）** | 回调5-20根K线 | 新手只做顺势；老手可双向 |
| **震荡区间（Trading Range）** | 持续≥20根K线，80%突破失败 | 高抛低吸，上1/3卖出，下1/3买入 |

强趋势很少直接变反向强趋势：通常是 强趋势 → 震荡区间 → 大反转（MTR）。

---

## 三、三种市场状态

### 趋势（Trending）
- HH+HL（多头）/ LH+LL（空头），多数K线收于极端
- 20 EMA 持续在价格下方（多头）或上方（空头）

**规则**：只做顺势；回调买/反弹卖；H1/H2 或 L1/L2 入场；止损在重要高低点外

### 震荡区间（Trading Range）
- 价格反复碰高低点边界，K线重叠、十字星多
- 20 EMA 横穿价格

**规则**：80%突破会失败→默认假设突破失败；高抛低吸快速止盈；Barbwire（极窄区间）→停止交易

### 过渡状态（Breakout Mode）
- 内含线增加，波动率收缩 → 停止交易，等方向确认

## 四、K线分类

| 类型 | 特征 | 含义 |
|------|------|------|
| **趋势K线** | 大实体、小影线、收于极端 | 单边控制 |
| **十字星/Doji** | 小实体、两端长影线 | 犹豫，需下根K线确认 |
| **内含线/Inside Bar** | 高低在前根K线内 | 蓄力，突破方向为信号 |
| **外包线/Outside Bar** | 吞没前根K线 | 剧烈双向博弈，收盘方向关键 |

---

## 五、H1/H2/L1/L2 入场框架

### 多头回调买入
| 信号 | 含义 | 可靠性 |
|------|------|--------|
| **H1** | 回调中第一根高点超过前一根 | 激进入场，弱趋势中易失败 |
| **H2** | 第二腿回调完成 | **黄金标准**，主要入场信号 |
| **H3** | 第三腿，通常形成楔形牛旗 | 高概率 |
| **H4** | 第四腿 | 罕见，趋势可能转向震荡 |

### 空头反弹卖出
| 信号 | 含义 |
|------|------|
| **L1** | 反弹中第一根低点低于前一根 |
| **L2** | 第二腿反弹完成，**主要做空信号** |
| **L3** | 楔形熊旗 |

### 入场机制
- 做多：H2 高点上方 1 tick 挂突破单；止损：回调最低点下方
- 做空：L2 低点下方 1 tick 挂突破单；止损：反弹最高点上方

## 六、顺势交易核心规则

### 回调与数K线
- 上涨趋势 = 低点不断抬高（比高点抬高更重要）
- 下跌趋势 = 高点不断降低（比低点降低更重要）
- 跌破重要低点 → 上涨趋势结束，大概率进震荡
- 突破重要高点 → 下跌趋势结束

### 20 EMA 关键规则
| 信号 | 条件 | 操作 |
|------|------|------|
| **M2B / M2S** | 第一次两腿回调触碰20 EMA | 顺势入场，最佳初学者信号 |
| **20 GAP Bar** | 价格在EMA一侧持续20+根K线后首次触碰 | 极高概率顺势入场 |

### Always In 原则
- 趋势未破坏 → 始终顺势持有
- 强突破：不等回调，市价入场
- 牛市阴线收盘买入 / 熊市阳线收盘卖出

---

## 七、突破交易规则

- 只有 5-10% 突破是真正趋势开端
- **强突破**：连续大实体、回调浅（1-2根）、与20 EMA迅速拉开→市价入场
- **弱突破/假突破**：小实体、长影线、无动能→按震荡区间处理
- 假突破回区间内部 → 反向交易机会

---

## 八、震荡区间交易规则

- 80% 突破失败规则：默认假设突破失败
- 区间确认需 2-3 次触碰边界
- 上 1/3 = 卖出区，下 1/3 = 买入区
- TRF（Trading Range Fade）入场，TRF2 概率更高
- 快速止盈，紧贴区间边界止损
- Barbwire（极窄重叠）→ 不交易

---

## 九、反转交易规则

### 大反转（MTR）四条件
1. 强趋势运行充分
2. 趋势线被突破
3. 回测前极端高低点
4. 反转信号K线确认入场

### 楔形反转（Wedge）
- 三次推进 = 可持续趋势通常在三推后修正
- 上升楔形→看跌；下降楔形→看涨
- 3P（Three Pushes）= 三推计数

### 抛物线反转（Climax）
- 趋势末端加速（3-4根急冲刺）
- 高潮后更可能进震荡，而非直接反转
- V-Top/V-Bot：极速反转

### 其他反转形态
| 形态 | 特征 |
|------|------|
| **Final Flag** | 趋势末端小旗形，突破失败后反转 |
| **Head & Shoulders** | 经典头肩顶/底 |
| **Double Top/Bottom** | 双顶/底后回调入场 |
| **Expanding Triangle** | 扩张三角形，5浪后反转 |
| **TCL Overshoot** | 突破趋势通道线后回归 |
| **2-Bar Reversal** | 强趋势K线+反向强趋势K线 |

## 十、风险管理

### 止损铁律
- 止损必须挂在市场中，不能只"心里有数"
- 做多止损 = 重要低点下方；做空止损 = 重要高点上方
- 突破行情止损在起涨/起跌K线极端位置

### 仓位管理
- 突破行情/大止损 → 小仓位；窄止损 → 正常仓位
- 永不全仓；盈利加仓只适合有经验者

### 盈亏比
- 1:1 → 需≥60%胜率；2:1 → 40%即可
- **盈利目标至少等于风险**

---

## 十一、早盘策略

1. 前 90 分钟最重要，确定当日基调
2. 高开：常回补缺口或延续；低开：常反弹回补或继续下跌
3. Breakout Mode = 早盘方向选择
4. 突破失败反转 = 早盘最常见反转类型

---

## 十二、交易心理学

- 收益率是结果不是目标；亏损必然发生
- 诚实面对错误；对手盘思维：理解对方为什么接单
- 多解性认知：多空都有理由，不要固执
- 职业交易员知道市场在做什么，所以没有情绪

---

## 十三、不同阶段速查

**强牛市（Always In Long）**：H1, H2, 20GAB, M2B；忽略卖出信号；止损信号K线低点下方

**震荡区间**：TRF/TRF2 高抛低吸；上1/3卖下1/3买；快速止盈

**反转捕捉**：MTR, WDG, V-TOP/BOT, Final Flag；等强信号K线确认；不做只因"涨太多了"的空
